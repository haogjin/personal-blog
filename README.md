#### 技术

vue.js+iview+node.js+espress+mysql

#### 运行步骤

``` bash
1、运行数据库文件myblog.sql

2、下载依赖：npm install

3、进入server文件夹目录运行node：node app

4、直接运行会进入用户查看界面：npm run dev

5、管理员界面入口：/login
   账号：Aimee   密码：111111
```
#### 说明
``` bash
1、用户：提供查看和搜索权限
2、管理员：登陆后可以对个人信息、文章、菜单进行增删改查操作
   注：动态菜单管理模块是根据自己订的规则进行添加修改的
```
